# สรุปข้อมูลตั้งต้นจากลูกค้า



## ชื่อโครงการ
- ให้บริการฝึกอบรมด้านความปลอดภัยและให้คำปรึกษาระบบมาตรฐานสากล

## บริบทธุรกิจ
- บริษัทรับจัดฝึกอบรม และให้คำปรึกษา ด้านความปลอดภัย สิ่งแวดล้อม และคุณภาพ  
โดย แบ่งบริการออกเป็น การจัดอบรม รูปแบบ Public และ In house  และให้คำปรึกษาระบบมาตรฐานสากลต่างๆ

## ปัญหาหลักที่ต้องการแก้
- การเก็บข้อมูล ประวัติการฝึกอบรม สถานที่อบรม บริษัทอบรม ข้อสอบ ใบลงทะเบียน รูปภาพ การออกวุฒิบัตร ไม่เชื่อมโยงกัน
การประสานงานระหว่างขั้นตอน และการติดตามงานกับลูกค้าและลงคิวอบรมผิดพลาด
ไม่สามารถ ดูสถานะงานได้  และไม่เห็นภาพรวมของธุรกิจ 
ต้องรอคอยการสรุปข้อมูลจากผู้ประสานงาน หรือแอดมิน

## เป้าหมายทางธุรกิจ
- ลดเวลาการทำงาน เพิ่มความแม่นยำสำหรับแผนงานกับทางลูกค้า การยืนยันงานกับวิทยากร การแนบหลักฐานการฝึกอบรม  การออกใบรับรองหลังอบรม  และระบบประมวลผล เชิงบริหาร รวมถึงข้อมูลประวัติต่างๆ

## ตัวชี้วัดความสำเร็จ
- 100% ที่ลูกค้า ติดต่อมา ต้องได้รับการตอบสนอง ออกใบเสนอราคา
- 100% ที่ลูกค้ายืนยัน ไม่มีการประสานงานผิดพลาด ทั้งวิทยากร และลูกค้า
- 100% ที่จัดฝึกอบรม ปิดงาน เก็บเงินได้ครบ
- 100% ของทุกหลักสูตรมีข้อมูลครบถ้วนที่สามารถสอบย้อนกลับได้

## กลุ่มผู้ใช้งาน
- ผู้จัดการบริษัท   ดูแลระบบองค์กร ทั้งงานอบรม และงานที่ปรึกษา
- เจ้าหน้าที่การตลาด  ประสานงาน ติดต่อลูกค้า ให้ได้ใบเสนอราคา ให้ได้รับการยืนยันจัดอบรม
- เจ้าหน้าที่ฝึกอบรม  จัดฝึกอบรมและสรุปผลการอบรม
- วิทยากร รับแจ้งจัดอบรม และไปสอน

## ผู้มีส่วนเกี่ยวข้องหลัก
- ผู้จัดการบริษัท   ดูแลระบบองค์กร ทั้งงานอบรม และงานที่ปรึกษา |  | medium
- เจ้าหน้าที่การตลาด  ประสานงาน ติดต่อลูกค้า ให้ได้ใบเสนอราคา ให้ได้รับการยืนยันจัดอบรม |  | medium
- เจ้าหน้าที่ฝึกอบรม  จัดฝึกอบรม แนบรูปภาพ ใบลงทะเบียน สร้างข้อสอบในระบบ และสรุปผลการอบรม  ออกวุฒิบัตร |  | medium
- วิทยากร รับแจ้งจัดอบรม และสอนแนบรูปภาพ ใบลงทะเบียน |  | medium
- ผู้เข้ารับการอบรม ทำแบบทดสอบในระบบ ก่อน-หลังอบรม |  | medium

## กระบวนการทำงานปัจจุบัน
- ใช้ manual operation ขึ้นอยู่กับบุคคล เช่นเก็บในคอมและ excel อาจจะสูญหาย หรือเก็บไม่ครบถ้วน
